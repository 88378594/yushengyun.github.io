# semo

>基于vue 开发的一个壁纸项目

## 运行步骤
## 项目初始化 安装依赖
npm install
## 运行项目
npm run dev

## 运行报错则执行
## 安装 rimraf
npm install rimraf -g
## 删除 node_modules以及package-lock.json
rimraf node_modules
rimraf package-lock.json
## 清除 项目缓存
npm cache clear --force

## 完成以上步骤再执行初始化
npm install
## 运行项目
npm run dev
